# Bug Reporting
To make a bug report, [open an issue](https://github.com/DraxFM/Flux/issues) on the GitHub Repo.
A bug report requires a well written explanation of expected behavior and actual behavior. Reproduction steps or a reproduction place would also be nice.

# Suggestions & Questions
Suggestions can also be made by [opening an issue](https://github.com/DraxFM/Flux/issues) at the GitHub Repo.  
For Questions, please ensure they are not answered within this documentation.
